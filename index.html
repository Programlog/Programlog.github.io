Work

Trading Analysis Platform

View on GitHub

This project is a comprehensive full-stack trading analysis platform built using the MERN stack (MongoDB, Express.js, React, Node.js). I designed and implemented advanced charting tools that allow users to visualize and analyze market trends with a variety of technical indicators. The platform integrates real-time market data feeds, enabling users to make informed decisions based on the latest information. One of the standout features is the integration of AI-powered price prediction models, which leverage machine learning algorithms to forecast potential market movements.

I led the architecture, backend API development, and the implementation of the AI modules, ensuring a seamless and responsive user experience. This project demonstrates my ability to combine data engineering, frontend design, and artificial intelligence to solve real-world financial problems.

Flappy Bird AI

View on GitHub

Flappy Bird AI is an artificial intelligence project where I developed a neural network agent capable of learning to play and master the Flappy Bird game autonomously. Using reinforcement learning techniques, the AI agent interacts with a custom-built game simulator, receiving feedback and improving its performance over thousands of iterations.

I implemented the neural network from scratch and designed the training loop to visualize the agent's progress in real time. This project not only highlights my understanding of machine learning concepts and Python programming, but also my ability to create engaging visualizations and simulations.

The result is an AI that can consistently achieve high scores, demonstrating the power of self-learning algorithms in game environments.

Weather Application

View on GitHub

This project is a modern iOS weather application, complemented by a web interface and hosted on Heroku. I engineered the app to fetch and display real-time weather data from public APIs, presenting users with up-to-date forecasts and current conditions. Additionally, the app aggregates relevant news articles to keep users informed about weather-related events. I focused on delivering a clean, intuitive, and visually appealing user interface, prioritizing both usability and aesthetics.

The project involved integrating multiple APIs, handling asynchronous data fetching, and designing for cross-platform compatibility. Through this work, I demonstrated my skills in mobile development, API integration, and user-centered design.
